Apa itu HTML?

HTML adalah bahasa markup untuk membuat halaman web.

Apa itu CSS?

CSS digunakan untuk mempercantik tampilan web.

Apa itu JavaScript?

JavaScript membuat web lebih interaktif.

Apa itu jQuery?

jQuery adalah library JavaScript yang memudahkan manipulasi DOM.

Apa itu DOM?

DOM adalah representasi struktur dokumen HTML.
